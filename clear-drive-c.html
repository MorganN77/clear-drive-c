วิธีเคลียร์พื้นที่ไดรฟ์ C อย่างปลอดภัย (Windows)

1. ลบ Temporary Files

  • กด Win + R พิมพ์ temp → ลบทั้งหมด
  • กด Win + R พิมพ์ %temp% → ลบทั้งหมด

2. ใช้ Disk Cleanup

  • ค้นหา "Disk Cleanup" → เลือก Drive C
  • คลิก “Clean up system files” → เลือกรายการเช่น:
    • Windows Update Cleanup
    • Previous Windows installations
    • Temporary files

3. ลบไฟล์ Prefetch

  • กด Win + R → พิมพ์ prefetch → ลบทั้งหมด

4. ลบ System Restore Point เก่า

  • ไปที่ Control Panel → System → System Protection → Configure
  • เลือก “Delete” หรือจำกัดขนาดพื้นที่

5. ลบ Windows.old

  • ใช้ Disk Cleanup → “Clean up system files” → เลือก “Previous Windows installations”

6. ลบไฟล์ Download ของ Windows Update

net stop wuauserv
net stop bits
del /s /q /f %windir%\SoftwareDistribution\Download\*
net start bits
net start wuauserv

7. ลบ Log และ Minidump

  • %SystemRoot%\Minidump
  • %SystemRoot%\MEMORY.DMP
  • %SystemRoot%\Logs

8. ล้าง Recycle Bin

  • คลิกขวาที่ถังขยะ → Empty Recycle Bin

9. ปิด Hibernate (ถ้าไม่ใช้)

powercfg -h off

ประหยัดพื้นที่ได้หลาย GB

10. ใช้ TreeSize Free เพื่อดูว่าโฟลเดอร์ใดใช้พื้นที่มาก

  • ดาวน์โหลดที่: https://www.jam-software.com/treesize_free
  • เปิด TreeSize แบบ Run as Administrator เพื่อดูข้อมูลครบ
